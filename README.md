# usual us 🤍

A private, cinematic Progressive Web App built by one developer for two people — a shared space
for expenses, photo memories, moods, milestones, music, and the little things worth remembering.
Vanilla HTML/CSS/JS · Firebase (Firestore) · Cloudinary · deployed on Vercel.
